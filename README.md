# TicketTracker

## Structure

- `backend/` — backend service (implementation pending)
- `frontend/` — frontend service (implementation pending)
- `docker-compose.yml` — orchestrates backend, frontend, and database containers
- `.env.example` — template for local environment configuration

## Getting started

1. Copy `.env.example` to `.env` and adjust values as needed.
2. Run `docker-compose config` to validate the compose file.

No business logic has been implemented yet — this is the initial project scaffold.
